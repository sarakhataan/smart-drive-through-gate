# Smart-drive-through-gate
An Arduino project to simulate an  automatic drive through gate.

## Circuit Design:
<img width="837" height="555" alt="Screenshot 2026-05-13 at 7 33 03 PM" src="https://github.com/user-attachments/assets/9f86d4e9-5f86-4b1d-8feb-e830eae464f0" />

### Componnets needed:
  ##### Arduino UNO
  ##### Ultrasonic distance sensor
  ##### LCD screen (i didn't use an i2c)
  ##### Servo motor
  ##### Potentiometer
  ##### 220 ohm resistor
  ##### Jumper wires
  ##### Bread board
