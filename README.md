# Smart-drive-through-gate
An Arduino project to simulate an  automatic drive through gate.

### Componnets needed:
  ##### Arduino UNO
  ##### Ultrasonic distance sensor
  ##### LCD screen (i didn't use an i2c)
  ##### Servo motor
  ##### Potentiometer
  ##### 220 ohm resistor
  ##### Jumper wires
  ##### Bread board
